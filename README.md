# 番茄小说创作库 (Fanqie Novel Creation Library)

> 都市爽文 · 开局即爆 · 打脸无敌

---

## 📖 关于本项目

这是一个专注于番茄小说网风格的短篇爽文创作仓库。所有作品均遵循番茄小说网的流行元素：快节奏、多爽点、标题党、无脑爽。

---

## 🎯 作品特点

- ✅ **节奏迅猛**：第一章觉醒系统，第二章开始逆袭
- ✅ **爽点密集**：每章都有打脸、升级、收获
- ✅ **元素齐全**：系统流、退婚流、商业打脸、古武修仙、星际科幻
- ✅ **标题炸裂**：自带流量，符合平台算法
- ✅ **无敌爽文**：主角一路开挂，反派全部打脸

## 🚀 创作理念

番茄小说网的成功公式：
```
标题党 + 快节奏 + 密集爽点 + 元素组合 + 无脑爽
= 高阅读量 + 高推荐 + 高收入
```

我们不追求文学性，只追求"爽"和"快"。读者用碎片时间阅读，需要持续的刺激和多巴胺释放。

---

## 📝 创作规范

1. **标题格式**：必须包含"开局""退婚""系统""震惊""逆袭"等热门关键词
2. **章节结构**：每章2000-3000字，结尾留钩子
3. **爽点密度**：每章至少1-2个小高潮，3章必有大打脸
4. **升级节奏**：主角实力/财富/地位快速增长，拒绝憋屈
5. **反派设定**：智商在线但遇到主角就倒霉
6. **女主安排**：1-3个女主，必须倒贴主角
7. **结局处理**：大团圆或平凡温馨，不能悲剧

## 🔧 如何使用

### 阅读作品
直接打开 `.md` 文件阅读即可，所有作品均为 Markdown 格式。

### 本地编辑
```bash
# 克隆仓库
git clone https://github.com/xf2008sy-cpu/shortstory.git

# 创作新作品
# 1. 复制模板文件
cp 番茄小说-开局被退婚我提取万亿资产震惊全城.md 你的新作品.md
# 2. 修改内容
vim 你的新作品.md
# 3. 提交推送
git add .
git commit -m "新增作品：你的作品名"
git push
```

## 🤝 贡献

欢迎投稿！投稿要求：
1. 符合番茄小说网风格（爽文、快节奏、标题党）
2. 字数 10,000-20,000 字
3. 包含完整故事情节（有头有尾）
4. 章节划分清晰（每章2000-3000字）

提交方式：
1. Fork 本仓库
2. 创建新分支 `your-novel`
3. 添加你的作品文件（命名格式：《标题》-.md）
4. 提交 Pull Request
5. 等待审核（通常1-3天）
